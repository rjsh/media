47/**
48 * @file fluidsynth.h
49 * @brief FluidSynth is a real-time synthesizer designed for SoundFont(R) files.
50 *
51 * This is the header of the fluidsynth library and contains the
52 * synthesizer's public API.
53 *
54 * Depending on how you want to use or extend the synthesizer you
55 * will need different API functions. You probably do not need all
56 * of them. Here is what you might want to do:
57 *
58 * - Embedded synthesizer: create a new synthesizer and send MIDI
59 * events to it. The sound goes directly to the audio output of
60 * your system.
61 *
62 * - Plugin synthesizer: create a synthesizer and send MIDI events
63 * but pull the audio back into your application.
64 *
65 * - SoundFont plugin: create a new type of "SoundFont" and allow
66 * the synthesizer to load your type of SoundFonts.
67 *
68 * - MIDI input: Create a MIDI handler to read the MIDI input on your
69 * machine and send the MIDI events directly to the synthesizer.
70 *
71 * - MIDI files: Open MIDI files and send the MIDI events to the
72 * synthesizer.
73 *
74 * - Command lines: You can send textual commands to the synthesizer.
75 *
76 * SoundFont(R) is a registered trademark of E-mu Systems, Inc.
77 */
